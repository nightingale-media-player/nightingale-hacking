Preferences

Main Page | Preferences

You can configure FolderSync / the folder synchronisation feature with a preference dialog available via the help section of the synchronisation tab and via the AddOn preferences when using FolderSync in Songbird. The preference dialog is diviced into two categories, this chapter will describe the options available there.

User interface

The first category determines the way folder synchronisation is integrated into the player's user interface.

Display global notifications

If enabled, you get notified with a global notification when a sync is completed, for example like this:

If you have global notifications enabled, you can restrict them by using "only if FolderSync / Synchronisation Tab is not opened" to only appear when you're not in the synchronisation tab; you recive a message about the completion of a sync there as well.

Enable Sync Favorites

If you disable this option, all UI elements for sync favorites will get hidden, so you have more space for the other UI elements, but you won't be able to use sync favorites anymore.

Your sync favorites won't get deleted when disabling sync favorites, after re-enabling you can access them again. You need to re-load the synchronisation tab for this option to apply.

Show Help section

If you disable this option, the Help section won't be displayed in the synchronisation Tab. You need to re-load the synchronisation tab for this option to apply.

This option is only avalable in FolderSync at the moment, as you need the help section in Nightingale to access the preference dialog.

Available Tags

In the second section of the preference dialog you can specify what values should be used for the tags in filename schema, when there is no metadata available for a song.

Albumartist's Name

If the option "Use Artist's Name if that tag is missing" is enabled, %albumartist% will get replaced by %artist% if there is not albumartist set in a song's metadata.
