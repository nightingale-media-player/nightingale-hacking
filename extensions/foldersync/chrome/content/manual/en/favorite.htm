Favorite Syncs

Main Page | Favorite Syncs

You can save syncs you use often as favorite sync, so you don't need to enter all sync-specific information every time you want to perform it. To work with favorite syncs it is needed you understood how to perform a sync.

Create or update a Favorite Sync

To create or update a favorite Sync, go to the synchronisation tab and set the sync to save as favorite up the same way as you would set up a normal sync. If you want to update a sync you might want to load the sync to get updated before doing the changes. Instead of clicking at the start sync button, you now use the button "Save as Favorite" to save the current sync as a new favorite sync, or you select a favorite sync to update in the favorite sync area at the top of the tab, and then take "Update Favorite Sync" to overwrite it with the current sync.

Load a Favorite Sync

To update a favorite sync or to perform a slightly different sync, you might want to load the favorite sync's values into the user interface. To do so, select the favorite sync you want to load in the favorite sync area at the top of the tab, and click the "Load" button on the right hand side of it.

Perform a Favorite Sync

To start a favorite sync, select it in the favorite sync area at the top of the tab, and choose "Start" on the right hand side of it.

Delete a Favorite Sync

If you don't need a favorite sync anymore, you can delete it by selecting it in the favorite sync area on the top and choosing the "Delete..." button on the right hand side of it.
