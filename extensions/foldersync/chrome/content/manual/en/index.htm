About This Manual

Folder synchronisation is available as AddOn for Songbird, FolderSync, and as built-in feature in Nightingale. There are no differences between those two versions; this manual applies to both.

Within this manual, brown links will guide you to other content, while blue links refer to additional content from the internet.

This manual contains some screenshots, those are taken from the FolderSync AddOn currently. Because the look of the UI differs depending on the feather (skin for Songbird / Nightingale) installed, each screenshot was made in "Purple Rain" (Songbird's default feather) and "simplyBlack". "Purple Rain"-Screenshots are on the left hand side or above (on small screens / browser windows).

On the bottom of each pages you can see what version the manual page relates to, it might happen that new features are available within the version you have installed. You can also see the language of the manual you're using; other languages' content might cover newer versions.
