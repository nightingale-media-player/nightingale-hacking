Detailed Description

This interface represents a transfer of a collection of media items and/or media lists.

This interface will be sent with the following flavors: application/x-sb-transfer-media-items (enumerator returns sbIMediaItem) application/x-sb-transfer-media-lists (enumerator returns sbIMediaList)

Note that since a media items is a media list, a transfer context whose flavor is "application/x-sb-transfer-media-items" may indeed still media lists. Therefore consumers who want to treat lists differently will need to check the contents of the enumerator to determine if lists are present.
